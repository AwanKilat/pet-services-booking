# MyPetCare

MyPetCare is a Flutter-based mobile application developed as a Final Year Project (FYP) at Universiti Tun Hussein Onn Malaysia (UTHM).

## Features

- User Authentication
- Pet Service Booking
- Appointment Management
- Payment Submission
- Booking History
- Customer Reviews
- Firebase Authentication
- Cloud Firestore Database

## Technologies

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
